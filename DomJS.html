Définition DOM

Le Document Object Model est une interface de programmation normalisée par le W3C,
qui permet à des scripts d'examiner et de modifier le contenu du navigateur web.
C'est aussi la structure d'une page web, ou d'autres équivalents.

Le premier élément d’un Dom sont les racines (Root), puis il y a les noeux HTML…
Qui utilise le système Parent/Enfant, pour ce reconnaître (en faîte on en parle plus içi, c’est tout).
C’est comme une représentation textuelle de Windows explorer.

									DOM Tutorial

					On y accède avec la propriété innerHTML.
					var myTitle = document.getElementById("demo").innerHTML;

					On y accède avec la propriété nodevalue du 1er enfant du nœud.
					var myTitle = document.getElementById("demo").firstChild.nodeValue;

					Équivalent de :
					var myTitle = document.getElementById("demo").childNodes[0].nodeValue;

					Pour ajouter un élément au Dom, il faut d’abord le créer, puis l’ajouter à son parent.
					Ne pas oublier que le texte est un textNode.
					Supprimer avec remove() ou removeChild().

					Très pratique :

					var child = document.getElementById("p1");
					child.parentNode.removeChild(child);

					Remplacer un élément par un autre avec :
					parent.replaceChild(para, child);

					Voir le damier en DOM.

					La propriété innerHTML permet d’accéder au contenu d’un paragraphe, Div….

					Document est à la racine de la page HTML ; il est le propriétaire de tous les objets de la page.

					On accède à tous nos objets par document. :

					document.getElementById(id)  

					document.getElementsByTagName(name)        

					document.getElementsByClassName(name)      

					Changer les éléments de la page :

					element.innerHTML =  new html content           

					element.attribute = new value 

					element.style.property = new style

					element.setAttribute(attribute, value)

					Ajout/Suppression d’éléments :

					document.createElement(element)        Create an HTML element

					document.removeChild(element)          Remove an HTML element

					document.appendChild(element)          Add an HTML element

					document.replaceChild(new, old)        Replace an HTML element

					document.write(text)
